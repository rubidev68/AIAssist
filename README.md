# KAIRA
## Conversational AI Assistant and Station based on GPT-4o

### Installation
#### 1.Clone repo
```bash
git clone https://github.com/rubidev68/KAIRA.git
cd KAIRA
```

#### 2.Install dependencies
```bash
npm install
```

#### 3. Add OpenAI api key
```bash
nano .env
#OPENAI_API_KEY=<your-api-key>
```

#### 4. Run
```bash
node base.js
```
